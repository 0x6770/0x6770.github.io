Run Ubuntu on Mac via Docker

2019-10-11 yujie wangenvironment

To ensure consistance across platforms, we were suggested to install ubuntu as vertual system in our first lecture on Programming. Dr David Thomas provides us with detailed solutions to get ubuntu on MacOS and Windows by VirtualBox and Hyper-V. But due to unknown reasons(probably lack of adaption to this latest OS version), VirtualBox often crashes on latest macOS Catalina. Here is an alternate way to run ubuntu on macOS.

  • You will be able to have a folder called “ubuntu” on your desktop which linked to /src/ in ubuntu.
  • You will be able to start ubuntu in about 1 second.
  • Docker container takes less resources on your computer than vm.
  • BUT no graphic user interface.

1. Download Docker (1.5 min)

Copy and paste the following link to the address bar of your preferred bowser would start the download of Docker-Desktop for Mac.

2. Install Docker (1 min)

  • Drag and drop Docker into Application folder. And wait for the process to complete.

  • When completed, you should be able to see Docker in your launchpad.

3. Start Docker (0.5 min)

  • Click Docker in the launchpad to start it. You should then be able to see Docker icon in the status bar.

  • Sign up and login into a docker account is not necessary. Just wait util it shows “Docker Desktop is running”.

Get Ubuntu run (1 min)

  • Open Spotlight Search ⌘ + SPACE and type “terminal” in the pop-up window.

  • Open Terminal.

  • Copy and paste the following code. Press Enter to run.

mkdir /Users/$USER/Desktop/ubuntu;
docker run \
  —name ubuntu \
  -e HOST_IP=$(ifconfig en0 | awk '/ *inet /{print $2}') \
  -v /Users/$USER/Desktop/ubuntu:/src \
  -t -I \
  ubuntu /bin/bash

  • If you see something like this, Ubuntu is already up and running.

  • You are now in the ubuntu. Check by run cat /etc/lsb-release

Install dependency and tools (0.5 min)

Continue in the previous window, run the following code one by one.

apt-get update
apt-get upgrade
apt-get install nano g++

Congratulations

Next time when you want to use ubuntu, just open terminal and run

docker start ubuntu; docker exec -it ubuntu /bin/bash

Make sure Docker is running before run the code.

P.S. You can create a new profile in terminal that run this code at start to save time.
